# 環境準備

本ハンズオンを進めるためには以下の環境が必要となります。

* Mac OS（SketchがMac OSのみ対応しているため）
* Sketch 最新版 (Version 52.3 を想定)
* ブラウザ（※IE不可、モダンブラウザ限定）
* [Titillium Web フォント](https://fonts.google.com/specimen/Titillium+Web) <br>※ Macへインストールお願いします。

上記環境が揃っていない場合は事前にインストール下さい。
